# Практика 12. Сетевой уровень
## RIP
В задании выполнены все три пункта. Сеть задается с помощью списков смежностей в формате `JSON` в [конфиг-файле](rip/src/main/resources/netConfig.json). Пример:
```Json
{
  "router2Connections": {
    "192.168.0.1": ["192.168.0.2", "192.168.0.3"],
    "192.168.0.2": ["192.168.0.1", "192.168.0.3"],
    "192.168.0.3": ["192.168.0.1", "192.168.0.2", "192.168.0.4"],
    "192.168.0.4": ["192.168.0.3"]
  }
}
```
Программа запускается командой
```
./gradlew rip:run
```
